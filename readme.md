# ETSIINF Archive

### English

> [!WARNING]
> Most of these projects are old, and use deprecated libraries or don't work or
> are unfinished. 
> Most of the code found in this repo is bad and I would like to think not 
> reflective of the code I (can) write today.

An archive of some of the projects I've made in my time at ETSIINF UPM.

Some of these projects included additional code provided by the school or by 
a subject's professor. I have tried NOT to include any such code as it does not
belong to me. If any traces of code were left, I am more than happy to remove
them. An issue to this repo or an email about it should be enough to let me
know.

### Spanish

> [!WARNING]
> La mayoría de estos proyectos son antiguos y utilzan librerías deprecadas
> o no funcionan o nunca los terminé.
> Además buena parte del código no es de gran calidad y me gustaría pensar que
> no representa el código que soy capaz de producir hoy.

Este repo contiene algunos de los proyectos que he hecho como parte de mi
grado en Ingeriería Informática en la ETSIINF UPM.

Algunos de estos proyectos incluian código de apoyo que no me pertenece, y 
por tanto, he intentado no incluirlo. Si he cometido algún error en este 
proceso, estoy más que dispuesto a quitar cualquier traza de código que no
haya escrito yo. Una issue en este repositorio o un correo son buenas formas
de comunicarmelo.
